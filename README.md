# Glampronto Test App

## Tech Stack

- Next.js
- React.js
- Typescript
- ESlint & Prettier

## Usage

Download the repository:

    git clone https://github.com/ajithpmohan/glampronto-test.git

## Build the App

```bash
npm install
# or
yarn install
```

## Starting App

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
